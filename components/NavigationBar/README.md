# Navigation Bar

The Navigation Bar component is a view composed of a left and right Button Bar and either a title
label or a custom title view.

## Installation

### Requirements

- Xcode 7.0 or higher.
- iOS SDK version 7.0 or higher.

### Installation with CocoaPods

To add the NavigationBar component to your Xcode project using CocoaPods, add the following to your PodFile:

~~~ bash
$ pod 'MaterialComponents/NavigationBar'
~~~

Then, run the following command:

~~~ bash
$ pod install
~~~
